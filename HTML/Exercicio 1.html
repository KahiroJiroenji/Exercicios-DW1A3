Lista de atividades para utilização das tags aeho

strong

Esta tag serve para dizer ao arquivo que o conteúdo em questão possui um valor de importancia forte.

em

Esta tag é utilizada para aferir mudança sutil de significado em uma sentença.

blockquote

É utilizado para citações longas. Como diria Um sabio chines:

我問了一個聖人，不同的是愛與友誼之間，他告訴我這個事實......愛更敏感，最安全的友誼。愛給了我們翅膀，友誼之地。在愛中，有更多的感情，在友誼的理解。愛是種植的並培養了感情，友誼來得厚顏無恥，和歡樂與悲傷交換，變得偉大而親愛的伴侶。但是，當愛是真誠的他帶著一位好朋友來當友誼是具體的，她充滿了愛和感情。當你有一個朋友或者是一種激情，兩種感情共存在你的心裡。

q

Por outro lado, Outros dizem que a tag q é Utilizada para citações mais curtas. E isto é muito bom.

abbr

Quem disse isso? A lider mundial que domina as táticas de Citações Abreviadas.

cite

Cito, citarei então Four Calendar Café do grupo americano Cocteau Twins.

dfn

A Tag DFN por sua vez, serve para demarcar a primeira definição de algum termo.

address
